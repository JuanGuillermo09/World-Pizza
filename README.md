# World Pizza Web

Sitio web desarrollado para un restaurante de pizzas, con un diseño moderno, atractivo y totalmente responsive. La aplicación permite presentar productos, promociones y contenido visual de forma dinámica, ofreciendo una experiencia agradable tanto en dispositivos móviles como de escritorio.

Fue construido utilizando HTML, CSS y JavaScript, y se encuentra desplegado en Firebase Hosting para su acceso en línea.
